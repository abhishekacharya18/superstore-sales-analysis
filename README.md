# superstore-sales-analysis
Title
Superstore Sales Data Analysis

Tools
Python, Pandas, Excel, Power BI

Dataset
Superstore Sales Dataset

Project Objective
Analyze sales performance, product profitability, and regional performance.

Data Cleaning Steps
• Removed duplicate rows
• Checked missing values
• Converted order date to datetime format
• Verified numeric columns

Analysis Performed
• Sales by category
• Profit by region
• Top 10 products by sales
• Discount impact on profit

Conclusion
Technology category records the highest sales.
West region generates the highest profit.
Several products generate negative profit.
Higher discounts reduce profit margins.
